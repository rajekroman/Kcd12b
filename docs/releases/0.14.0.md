# 0.14.0 — Alchemy and Smithing

## Přidáno

- Anežčin kontextový bylinkářský stůl.
- Bohdanova kontextová kovárna.
- Pět datově definovaných receptů.
- Bylinný obklad s léčením 28.
- Bylinkami konzervovaná zvěřina s léčením 10.
- Kožené řemeny jako řemeslný mezimateriál.
- Kalený Bohdanův meč s útokem +8.
- Vyztužená prošívanice se zbrojí +7.
- Klávesa C a mobilní tlačítko Řemeslo.
- Samostatný responzivní crafting panel.
- Runtime atributy dostupnosti, stanice, zprávy a posledního receptu.

## Technické změny

- Nový čistý `CraftingSystem` bez DOM a Phaser závislostí.
- Atomické odebrání všech surovin a přidání všech výstupů.
- Kontrola dostupného množství, stack limitů a nosnosti.
- Bezpečné uvolnění equipment slotu při spotřebování vybavené položky.
- Propojení s EconomyStore a existujícím `ECONOMY_CHANGED` save řetězcem.
- Žádná změna save formátu; výrobky se ukládají v ekonomické části verze 5.
- Capture keyboard handler blokující konfliktní inventářové a bojové vstupy.
- Pozastavení a obnovení Arcade Physics a Phaser keyboard inputu.

## Testování

- Jednotkové testy úplnosti receptů a stanic.
- Testy atomiky při chybějící surovině a plném stacku.
- Test automatického sundání spotřebovaného meče.
- Desktopové a mobilní E2E Anežčiny alchymie.
- Desktopové a mobilní E2E Bohdanovy kovárny.
- Kontrola viditelného výstupu, statistik, IndexedDB save a reloadu.
- Regrese pokusu otevřít panel mimo řemeslnou stanici.
